# Modern Lending Nevada

Official website for Derek Lapikas and the Modern Lending team serving Las Vegas and Henderson, Nevada.

## Features
- Educational loan program guides (Conventional, FHA, VA, USDA, Jumbo, Refinance)
- Interactive Refinance Savings Calculator
- Value-first content to build trust immediately
- Netlify Forms for lead capture
- Fully SEO-optimized (schema, local keywords, E-E-A-T structure)
- Ready for blog expansion

## Local Focus
- Las Vegas & Henderson specific guidance
- Partnerships with local real estate professionals

## Deployment
This site is built for **Netlify**.

### Quick Deploy
1. Push this repo to GitHub
2. Go to [Netlify](https://app.netlify.com) → New site from Git
3. Connect this repository
4. Deploy settings:
   - Build command: (leave blank)
   - Publish directory: `.` (or `dist` if you add a build step later)
5. Forms will work automatically

### Custom Domain
Add your domain in Netlify → Domain settings.

## Updating the Site
- Edit `index.html` directly
- Commit and push — Netlify auto-deploys
- For bigger changes, we can migrate to Next.js + Contentlayer for a proper blog
